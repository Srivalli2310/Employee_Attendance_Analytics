# Employee Attendance Analytics

This project analyzes employee attendance data using Python and Pandas.

## Features
- Calculates present, absent, and late days
- Computes attendance percentage
- Identifies low attendance employees
- Visualizes attendance using a bar chart

## Technologies Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## How to Run
1. Open the notebook in Jupyter
2. Run cells sequentially
3. View analytics and visualizations

## Output
- Attendance summary table
- Attendance percentage bar chart
- CSV report file
